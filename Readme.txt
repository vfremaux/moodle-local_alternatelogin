Provides a compact management of an alternate login page.

the login form is a 2 panel/steps form collecting one (more in the future) profilefield.
It ends up in registering users on the email auth. 

Setup
##############

Unzip to local directory and install the plugin in admin notifications.

Go to Site admin/plugins/authentication main page.
Enable the email authentication.
Set the alternate login page to : <yourwwwroot>/local/alternatelogin/login.php
